My-AI-Chatbot is a modern, production-ready AI chat application built with Next.js, React, and TypeScript, designed to deliver fast, intelligent, and interactive conversations. It leverages the Vercel AI SDK and Google Gemini 2.5 Flash, along with support for multiple leading AI providers, to provide real-time streaming responses with a smooth and immersive user experience.

The project focuses heavily on mobile-first responsiveness, clean component architecture, and advanced UI/UX design, featuring glassmorphism, neon glow effects, and fluid animations powered by Framer Motion. It includes essential chatbot capabilities such as multi-conversation management, voice input, streaming responses, and a scalable frontend structure suitable for real-world applications.
